## 🌐 Deskripsi

Personal website template dengan desain modern dan responsif. Template ini cocok untuk portofolio profesional, blog personal, atau halaman tentang diri Anda.

## ✨ Fitur

- ✅ **Desain Modern & Responsif** - Tampilan profesional yang menyesuaikan dengan semua ukuran layar
- ✅ **Dark Theme** - Tema gelap dengan gradient accents yang menarik
- ✅ **Smooth Scroll Navigation** - Navigasi halus saat scroll ke section
- ✅ **Mobile-Friendly** - Dilengkapi hamburger menu untuk tampilan mobile
- ✅ **5 Sections Lengkap** - Hero, About, Skills, Projects, Contact
- ✅ **Animasi Hover** - Transisi halus pada button, card, dan link
- ✅ **Fully Customizable** - Mudah disesuaikan dengan kebutuhan Anda

## 📁 Struktur Proyek

```
├── index.html          # Halaman utama

```

## 🚀 Cara Penggunaan

### 1. Clone Repository
```bash
git clone https://github.com/antono4/macan.github.io.git
cd macan.github.io
```

### 2. Kustomisasi
Edit file `index.html` untuk menyesuaikan:

| Section | Isi yang dapat diedit |
|---------|---------------------|
| **Hero** | Nama, tagline, button text |
| **About** | Deskripsi diri, foto profil |
| **Skills** | Keahlian dan teknologi |
| **Projects** | Judul, deskripsi, tags proyek |
| **Contact** | Link social media |
| **Footer** | Credit dan attribution |

### 3. Deploy ke GitHub Pages
1. Push ke repository GitHub Anda
2. Buka **Settings** > **Pages**
3. Pilih branch `main` dan folder `/ (root)`
4. Website akan tersedia di `https://username.github.io/repo-name`

## 🎨 Kustomisasi Warna

Edit CSS variables di `:root` untuk mengubah warna tema:

```css
:root {
    --primary-color: #6366f1;      /* Warna utama */
    --secondary-color: #8b5cf6;    /* Warna sekunder */
    --bg-dark: #0f172a;            /* Background utama */
    --bg-card: #1e293b;            /* Background card */
    --text-primary: #f1f5f9;       /* Warna teks utama */
    --text-secondary: #94a3b8;     /* Warna teks sekunder */
    --gradient: linear-gradient(135deg, #6366f1 0%, #8b5cf6 50%, #d946ef 100%);
}
```

## 📱 Responsivitas

Template ini sudah dioptimalkan untuk berbagai ukuran layar:

| Device | Breakpoint | Fitur |
|--------|------------|-------|
| Desktop | 1200px+ | Layout penuh |
| Tablet | 768px - 1199px | Grid adjustment |
| Mobile | 320px - 767px | Single column, hamburger menu |

## 🛠️ Tech Stack

- **CSS3** - Styling modern dengan Flexbox, Grid, dan CSS Variables
- **JavaScript** - Interaktivitas vanilla (tanpa library)
